## About

Classic arcade games collection with retro favorites.

## Features

- Multiple classic games
- High score tracking
- Retro graphics
- Keyboard controls

## Installation

### From App Store
Open the **App Store** in zOS and search for "Arcade".

### Via npm
```bash
npm install @zos-apps/arcade
```

## Usage

- Select a game to play
- Use arrow keys for movement
- Check game-specific controls



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
